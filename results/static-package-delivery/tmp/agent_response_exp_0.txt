Below is a minimal PDDL specification that captures exactly the knowledge that was given.  
No additional assumptions are made – only the two locations that actually exist (the post‑office and the house) are used, and the truck can only move between these two locations.  

The domain defines the four objects that are needed, the predicates that describe the truck’s position, the package’s position, and whether the package is inside the truck.  
The planning problem states the initial situation that the truck and the package are both at the post‑office and declares the goal that the package is at the house.  

```pddl
;; =====================  Domain  =====================

(define (domain truck)
  (:requirements :typing :strips)

  ;; ----------------------------------------------------
  ;; Types
  (:types truck package location)

  ;; ----------------------------------------------------
  ;; Predicates
  (:predicates
    (at      ?t - truck    ?l - location)   ; truck is at a location
    (at-pkg  ?p - package  ?l - location)   ; package is on the ground at a location
    (in      ?p - package  ?t - truck)      ; package is inside the truck
  )

  ;; ----------------------------------------------------
  ;; Actions
  (:action move-to-house
    :parameters (?t - truck)
    :precondition (at ?t post-office)
    :effect (and
              (at ?t house)
              (not (at ?t post-office)))
  )

  (:action move-to-postoffice
    :parameters (?t - truck)
    :precondition (at ?t house)
    :effect (and
              (at ?t post-office)
              (not (at ?t house)))
  )

  (:action load
    :parameters (?t - truck ?p - package ?l - location)
    :precondition (and
                    (at ?t ?l)
                    (at-pkg ?p ?l))
    :effect (and
              (in ?p ?t)
              (not (at-pkg ?p ?l)))
  )

  (:action unload
    :parameters (?t - truck ?p - package ?l - location)
    :precondition (and
                    (at ?t ?l)
                    (in ?p ?t))
    :effect (and
              (at-pkg ?p ?l)
              (not (in ?p ?t)))
  )
)
```

```pddl
;; =====================  Problem  =====================

(define (problem deliver-package)
  (:domain truck)

  ;; ----------------------------------------------------
  ;; Objects in the world
  (:objects
    truck1     - truck
    package1   - package
    post-office house   - location
  )

  ;; ----------------------------------------------------
  ;; Initial situation
  (:init
    (at truck1 post-office)    ; truck starts at the post‑office
    (at-pkg package1 post-office)   ; the package starts at the post‑office
  )

  ;; ----------------------------------------------------
  ;; Goal
  (:goal (at-pkg package1 house))
)
```
