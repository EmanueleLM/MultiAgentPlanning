## `delivery-domain.pddl`

```pddl
(define (domain delivery)

  ; --------------------------------------------------------------------
  ; Types
  ; --------------------------------------------------------------------
  (:types
    vehicle   ; only the truck
    package   ; only the single package
    location  ; the post office and the house

    ; the top‑level type is implicit – objects of the above types are
    ; considered “objects” by PDDL.
  )

  ; --------------------------------------------------------------------
  ; Predicates
  ; --------------------------------------------------------------------
  (:predicates
    ; The truck, the package and the locations.
    (at ?obj - object ?loc - location)

    ; “in” indicates that a package is inside the truck.
    (in ?pkg - package)
  )

  ; --------------------------------------------------------------------
  ; Actions
  ; --------------------------------------------------------------------
  (:action drive
    :parameters (?v   - vehicle
                ?from - location
                ?to   - location)
    :precondition (and
                   (at ?v ?from))
    :effect (and
             (not (at ?v ?from))
             (at ?v ?to))
  )

  (:action load
    :parameters (?loc - location)
    :precondition (and
                   (at truck ?loc)
                   (at pkg ?loc))
    :effect (and
             (not (at pkg ?loc))
             (in pkg))
  )

  (:action unload
    :parameters (?loc - location)
    :precondition (and
                   (at truck ?loc)
                   (in pkg))
    :effect (and
             (not (in pkg))
             (at pkg ?loc))
  )
)
```

## `delivery-problem.pddl`

```pddl
(define (problem deliver-package)
  (:domain delivery)

  ; --------------------------------------------------------------------
  ; Objects
  ; --------------------------------------------------------------------
  (:objects
    truck - vehicle
    pkg   - package

    postoffice - location
    house      - location
  )

  ; --------------------------------------------------------------------
  ; Initial state
  ; --------------------------------------------------------------------
  (:init
    (at truck postoffice)
    (at pkg   postoffice)
    ; “in” is false by default – we do not need to mention it.
  )

  ; --------------------------------------------------------------------
  ; Goal
  ; --------------------------------------------------------------------
  (:goal
    (at pkg house))
)
```

### The optimal plan for this problem

```
1. (load truck pkg postoffice)
2. (drive truck postoffice house)
3. (unload truck pkg house)
```

This plan uses only the objects and predicates that were explicitly given, satisfying the goal that the package ends up at the house.
